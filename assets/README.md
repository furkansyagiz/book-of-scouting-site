# Assets Guide

Place your real images in these folders and then update image paths in `index.html`.

## Folder Structure

- `book/` - Book cover and related visuals
- `author/` - Author portrait images
- `testimonials/` - Testimonial speaker photos
- `events/` - Media and launch event photos
- `collaborators/` - Collaborator logos

## Notes

- Keep image widths at least 1200px for hero/event images.
- Use compressed JPG/WebP for photos and PNG/SVG for logos.
- Use lowercase filenames with hyphens for consistency.
